# Carpool Match Review MVP

A React + TypeScript + TailwindCSS prototype that allows users to review carpool recommendations one at a time in a dating-app style interface.

## Features

- View one recommendation at a time
- View:
  - Name
  - Employer
  - Start Location
  - End Location
  - Distance from start point
  - Distance from end point
- Thumbs Up button advances to next recommendation
- Thumbs Down button advances to next recommendation

## Tech Stack

- React
- TypeScript
- TailwindCSS
- Vite
